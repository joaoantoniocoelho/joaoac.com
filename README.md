# 🌌 joaoac.com

Welcome to my personal portfolio website! 🚀 Dive into an interactive 3D space battle scene, explore real-time Medium blog posts, and discover my professional journey. This site is a testament to the power of AI pair programming, crafted with Claude 3.5, Cursor IDE, and Bolt.new.

## 🌟 Overview

A modern, responsive portfolio that blends creative 3D visualizations with professional content. Featuring a unique space-themed hero section with interactive battles, a clean presentation of my work experience, and seamless Medium blog integration.

## ✨ Key Features

- **🚀 Interactive 3D Space Scene**
  - Dynamic space battles with shooting effects
  - Animated galaxy background

- **👨‍💻 Professional Sections**
  - About me overview
  - Work experience timeline
  - Real-time Medium blog posts
  - Professional contact information

## 🛠️ Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **3D Graphics**: Three.js & React Three Fiber
- **Styling**: Tailwind CSS
- **Components**: shadcn/ui
- **Animations**: Framer Motion
- **Blog Integration**: Medium RSS Feed
- **Icons**: React Icons

## 🛠️ Development

This project was developed using:
- Claude 3.5 for AI pair programming
- Cursor IDE for enhanced development
- Bolt.new for project management

*This project demonstrates that AI-assisted development can produce professional-quality applications while maintaining clean code standards.*

## 🎨 Vibe Code

This portfolio was built entirely through AI pair programming, without writing a single line of code manually. Even this README and the information within were crafted by AI. I sent my CV to GPT-4o, which generated the necessary JSONs and information. It's designed to captivate and engage, reflecting my passion for technology and storytelling. Whether you're exploring the cosmos or reading about my latest projects, I hope you feel inspired and connected. 🌌✨
