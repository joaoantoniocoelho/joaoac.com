# joaoac.dev

My personal portfolio website featuring an interactive 3D space battle scene, real-time Medium blog integration, and a showcase of my professional experience. Built entirely through AI pair programming using Claude 3.5, Cursor IDE, and Bolt.new.

## Overview

A modern, responsive portfolio website that combines creative 3D visualizations with professional content. The site features a unique space-themed hero section with interactive battles, a clean presentation of my work experience, and integration with my Medium blog posts.

## Key Features

- **Interactive 3D Space Scene**
  - Dynamic space battles with shooting effects
  - Animated galaxy background
  - Mouse-responsive particle system

- **Professional Sections**
  - About me overview
  - Work experience timeline
  - Real-time Medium blog posts
  - Professional contact information

## Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **3D Graphics**: Three.js & React Three Fiber
- **Styling**: Tailwind CSS
- **Components**: shadcn/ui
- **Animations**: Framer Motion
- **Blog Integration**: Medium RSS Feed
- **Icons**: React Icons

## Development

This project was developed using:
- Claude 3.5 for AI pair programming
- Cursor IDE for enhanced development
- Bolt.new for project management

*This project demonstrates that AI-assisted development can produce professional-quality applications while maintaining clean code standards.*
